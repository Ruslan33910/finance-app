  • ⚡ Быстрый ввод трат в пару касаний
  • 📊 Отчёт: куда уходят деньги
  • 🎯 Лимиты по категориям и цели накопления
  • 📴 Работает офлайн, ставится как приложение
  • 🔒 Данные хранятся только у вас на устройстве

Как установить

Android (Chrome): откройте приложение → меню «⋮» → «Установить приложение».

iPhone (Safari): откройте приложение → «Поделиться» → «На экран Домой».

Компьютер: в адресной строке нажмите значок установки.

Бесплатно. Без регистрации. Приложение не собирает и не отправляет ваши данные — всё остаётся на вашем устройстве.
